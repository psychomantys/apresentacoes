O comando “test”

  • Testa uma condição, retornando erro se falso.

    • test CONDIÇÃO

  • Exemplos:

    • test “” #retorna erro, falso. string vazia é falso.
    • test “qualque coisa” #verdade. Não esta vazio.
    • test “${x}” = “${x}” #verdade.
